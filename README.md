# JDMemberCloseLinks

本项目大部分代码基于 https://github.com/yqchilde/JDMemberCloseAccount （MIT协议开源）

## 使用方法

1. 克隆到本地

    ```shell
    git clone https://github.com/hyzaw/JDMemberCloseLinks.git
    ```

2. 安装所需要的包



    ```shell

    pip install -r requirements.txt

    ```

3. 运行程序

    ```shell

    python main.py
    ```
